# AVCoders.SignalR.Volume

SignalR hub and bridge for AV Coders volume controls. Part of the [AV Coders device library](https://github.com/AV-Coders/c-sharp-device-library). Targets **.NET 8.0**.

## Install

```bash
dotnet add package AVCoders.SignalR.Volume
```

Available from the AV Coders GitHub Packages feed (`https://nuget.pkg.github.com/AV-Coders/index.json`) and nuget.org. MIT licensed.

## What's inside

- `IVolumeHub`
- `VolumeUiSignalR` - indexed level and mute over a list of `VolumeControl`
- `VolumeManager`
