# AVCoders.SignalR.Room

SignalR hub and bridge for room state. Part of the [AV Coders device library](https://github.com/AV-Coders/c-sharp-device-library). Targets **.NET 8.0**.

## Install

```bash
dotnet add package AVCoders.SignalR.Room
```

Published to [nuget.org](https://www.nuget.org/packages/AVCoders.SignalR.Room). See the [repository README](https://github.com/AV-Coders/c-sharp-device-library) for details.

## What's inside

- `IRoomHub`
- `RoomUiSignalR` - room power on/off plus a string property bag
- `RoomManager`
