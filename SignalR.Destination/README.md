# AVCoders.SignalR.Destination

SignalR hub and destination definitions for AV routing UIs. Part of the [AV Coders device library](https://github.com/AV-Coders/c-sharp-device-library). Targets **.NET 8.0**.

## Install

```bash
dotnet add package AVCoders.SignalR.Destination
```

Available from the AV Coders GitHub Packages feed (`https://nuget.pkg.github.com/AV-Coders/index.json`) and nuget.org. MIT licensed.

## What's inside

- `IDestinationHub`
- `DestinationDefinition` - name, id, icon, routed source and video mute state
- `DestinationUiSignalR` and `DestinationManager`
